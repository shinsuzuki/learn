CHAPTER 1
Reactプロジェクトの作成（対話形式）
    npm create vite@latest study-react-app


必要なパッケージをインストール
    npm install


プロジェクトフォルダに移動
    cd study-react-app（プロジェクトフォルダ名）


開発サーバーの起動
    npm run dev

CHAPTER 11
JSON Serverのインストール
    npm install -g json-server

JSON Serverのアンインストール
    npm uninstall -g json-server

JSON Serverの起動
    npx json-server --watch db.json --port 3001

Axios のインストール
    npm install axios
